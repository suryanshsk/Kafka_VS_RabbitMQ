# Kafka vs RabbitMQ Interactive Whiteboard

This repository contains the interactive teaching whiteboard comparing Kafka and RabbitMQ.

## Files
- `kafka_vs_rabbitmq_interactive_whiteboard.html`: Full interactive whiteboard in dark mode.

## Highlights
- Full dark mode theme for better classroom visibility.
- Interactive sections for architecture, push vs pull, retention, broker model, use cases, and decision tool.
- Uses RabbitMQ and Kafka logos in the UI.

## How to use
Open `kafka_vs_rabbitmq_interactive_whiteboard.html` in any modern browser.
